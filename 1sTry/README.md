1,模块和类
1.1模块
模块类似于将一种东西对应到另外一种的方式，即“从Y获取X"的概念，主要特性如下：
1，模块是包含函数和变量的Python文件
2，可以导入这个文件
3，然后可以使用.操作符访问模块中的函数和变量。
1.2类
类和模块差不多，理解为俄罗斯套娃吧，一个大的套着小的，然后再小的包。。
